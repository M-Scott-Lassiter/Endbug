# Entity Documentation -  minecraft:fire_immune

`minecraft:fire_immune` allows an entity to take 0 damage from fire.

## Example

```json
"minecraft:fire_immune":{
}
```

## Vanilla entities examples

### magma_cube

```json
"minecraft:fire_immune": {
}
```
